# C# .NET Windows Forms Projects Collection

A collection of **C# .NET Windows Forms** projects developed using **Visual Studio 2026**, showcasing my practical implementation of the concepts learned throughout my C# journey.

This repository focuses on applying **Windows Forms controls**, **event-driven programming**, **object-oriented programming (OOP)**, and **Clean Code practices** by building complete desktop applications with organized and reusable code.

Each project was built as a hands-on exercise to strengthen my understanding of the .NET ecosystem and desktop application development.

---

# 🧱 Project Architecture

All projects follow the **Divide & Conquer** principle by separating responsibilities into small, reusable methods instead of placing all logic inside event handlers.

The applications emphasize:

- Clean Code
- Readability
- Maintainability
- Reusable functions
- Organized event handling
- Separation between UI and business logic whenever applicable
